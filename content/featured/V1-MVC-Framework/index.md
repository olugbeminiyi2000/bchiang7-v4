A lightweight MVC framework built from scratch in Python with no external web frameworks. Features a custom HTTP server on raw sockets, a router with persistent state, a model layer with atomic writes and transactions, built-in key/value validation, a template engine, and static file serving. Deployed with a live documentation site.
